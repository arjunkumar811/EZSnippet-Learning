HTML Image Tag

The <img> tag is used to embed images in an HTML document. It is a self-closing tag, meaning it doesn't need a closing tag.

Basic Image Syntax

<img src="image-path.jpg" alt="Description of the image">

Result:

Essential Attributes

Attribute Description Required?
src Specifies the path to the image file Yes
alt Provides alternative text if the image cannot be displayed Yes (for accessibility)
width Specifies the width of the image No
height Specifies the height of the image No
title Adds a tooltip when hovering over the image No
loading Specifies if the image should be loaded immediately or lazy-loaded No

Image with Width and Height

<img src="image.jpg" alt="Sample image" width="300" height="200">

Note: Setting the width and height helps the browser allocate space for the image before it loads, reducing layout shifts.

Responsive Images

Using CSS

<img src="image.jpg" alt="Responsive image" style="width:100%; height:auto;">

Result:

This image will resize to fit the container width (try resizing your browser)

Using srcset Attribute (Advanced)

<img src="small.jpg" alt="Responsive image with srcset" srcset="small.jpg 500w, medium.jpg 1000w, large.jpg 1500w" sizes="(max-width: 600px) 500px, (max-width: 1200px) 1000px, 1500px">

This allows browsers to choose the most appropriate image size based on the device's screen size and resolution.

Common Mistakes to Avoid

  • Not providing an alt attribute
  • Using very large images without optimization
  • Forgetting to specify width and height
  • Using the wrong image format for the content type
  • Not making images responsive for mobile devices

Warning: Large, unoptimized images can significantly slow down your website, affecting user experience and SEO rankings.
