Self-Closing HTML Tags

Self-closing tags (also known as void elements) are HTML elements that don't require closing tags. Here are some examples:

1. Image Tag <img>

<img src="image.jpg" alt="Sample image">

2. Line Break <br>

<p>This is a line of text.<br>This text appears on a new line.</p>

4. Input Fields <input>

<input type="text" placeholder="Enter your name">
<input type="password" placeholder="Enter password">
<input type="checkbox"> Remember me

5. Meta Tags <meta>

<meta name="description" content="This is a description of the page">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

6. Link Tag <link>

<link rel="stylesheet" href="styles.css">

Result: (Not visible, but used to link external stylesheets)

Practice Exercise

Try creating your own HTML page that uses at least 3 different self-closing tags in meaningful ways.

Notes:

  • In HTML5, the trailing slash is optional: <img src="image.jpg">
  • In XHTML, the trailing slash is required: <img src="image.jpg" />
  • Always include necessary attributes (like src and alt for images)
